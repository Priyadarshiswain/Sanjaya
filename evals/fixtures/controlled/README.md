# SignalDesk

SignalDesk is a synthetic incident-routing and notification project created
solely for Sanjaya's public evaluation suite. It is not a production
application and contains no private or migrated source.

The fixture deliberately includes:

- C# declaration ambiguity, overloads, partial types, and comment/string
  decoys;
- TypeScript and JavaScript application structure;
- generic configuration and operational documentation;
- deterministic Git history and working-tree changes created by the fixture
  builder; and
- generated scale profiles used only for traversal and limit testing.

The project is licensed with the rest of Sanjaya under Apache-2.0.
